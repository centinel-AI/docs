# Dashboard

The centinelAI dashboard shows the real-time status
of all your monitored infrastructure.

## Main sections

<div class="grid cards" markdown>

-   :material-view-dashboard:{ .lg .middle } **Services**

    ---

    ServiceCards with status (UP/DEGRADED/DOWN), risk score
    and sparklines of activity from the last 2 hours.

    [:octicons-arrow-right-24: View guide](services.md)

-   :material-alert-circle:{ .lg .middle } **Incidents**

    ---

    Full incident management with list view and timeline.
    Automatic postmortem on resolution.

    [:octicons-arrow-right-24: View guide](incidents.md)

-   :material-archive:{ .lg .middle } **Alerts archive**

    ---

    Complete history of all received alerts
    with filters by severity, source and score.

    [:octicons-arrow-right-24: View guide](alerts-archive.md)

</div>

## Real-time stats

| Metric | Description |
|--------|-------------|
| **Alerts today** | Total alerts received today |
| **Filtered** | Alerts absorbed without interrupting the team |
| **Interruptions sent** | Notifications sent to Slack/Email |
| **Open incidents** | Currently active incidents |
